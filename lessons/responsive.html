Responsiveness and Flexbox

What is Flexbox?

Flexbox is a one-dimensional layout method for laying out items in rows and columns. When you set the display property to flex, you get a main axis and a cross axis. The flex-direction sets how flex items are placed in the flex container defining the main axis and the direction. The justify-content property defines how the browser distributes space between and around content items along the main-axis of a flex container, and the inline axis of a grid container.
