Responsiveness and Flexbox

What is Flexbox?

Flexbox is a one-dimensional layout method for laying out items in rows and columns. When you set the display property to flex, you get a main axis and a cross axis. The flex-direction sets how flex items are placed in the flex container defining the main axis and the direction. The justify-content property defines how the browser distributes space between and around content items along the main-axis of a flex container, and the inline axis of a grid container. The flex-wrap property sets whether flex items are forced onto one line or can wrap onto multiple lines. Align axis sets the align-self value on all direct children as a group. In flexbox, it controls the alignment of items on the Cross Axis. The Align Content property sets the distribution of space between and around content items along the flexbox's cross-axis or a grid's block axis. The align-self property overrides a grid or flex item's align-items value.

  • Flex-Basis: Defines the initial size of an element before additional space is distributed.
  • Flex-grow: Controls the amount of available space an element should take up. Accepts a unit-less number value.
  • Flex-Shrink: If items are larger than the container, they shrink according th flex-shrink.
  • Flex is the shorthand for all 3 properties. flex-Grow | flex-basis | flex-shrink
