CSS

Cascading Style Sheets is a language for describing how documents are presented visually [Styled and Arranged].

Basic Patter: Selector { property: value; }

You can write your styles inline ass attributes for elements.You can put a style element in the head of the document. The best approach is to write an external CSS file and connect it to your HTML files. The way you connect a CSS file to an HTML file is using the link element.

The color property will change the color of the text that it's selecting. The background-color property will change the background color of an element.

Colors:

  • RGB - rgb(0 [red], 0 [green], 0 [blue]) COLOR
  • Hex (base 16) - #ffff00 COLOR

Text Properties

  • Text Align - Used to set the horizontal alignment of content inside an element or block element.
  • Font-weight - Used to control the weight/boldness of text.
  • Text-decoration - Controls the appearance of decorative lines on text.
  • line-height - controls the height of a line.
  • line-spacing - controls the spacing between text.
  • font-size - changes the size of the font of text in an element
  • font-family

Selectors

  • * - Universal Selector
  • Element Selector- Will select all elements of a given type
  • You can select multiple elements using a comma
  • # - An id selector
  • . - the class selector
  • descendant selectors are selected with spaces
  • + - used to select the adjacent element
  • > - used to select the direct child of an element
  • [] - selects the specific attribute of an element

Pseudo Classes

A pseudo class is a keyword added to a selector that specifies a special state of the selected element(s)

  • :hover
  • :active
  • :checked
  • :nth-of-type

Pseudo Elements

A keyword added to a selector that lets you style a particular part of selected element(a)

  • ::first-letter
  • ::first-line
  • ::selection

The Cascade and Specificity

The order your styles are declared in and linked to matters. Specificity is how the browser decides which rules to apply when multiple rules could apply to the same element. It is a measure of how specific a given selector is. The more specific wins. ID > CLASS > ELEMENT
